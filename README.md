## 山商论坛
学习 spring boot
## 部署
### 依赖
- Git
- JDK
- Maven
- MySQL
### 步骤
- yum update
- yum intstall git
- mkdir App
- cd App
- git clone https://github.com/Yangyang176/community.git
- yum install maven
- mvn -v
- mvn compile package
